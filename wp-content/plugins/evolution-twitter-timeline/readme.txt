== Description ==

Creates a new and simple to use widget that outputs the awesome Twitter Embedded Timeline. 
Localizations: German, English

== Installation ==

1. Upload contents of the directory to /wp-content/plugins/ (or use the automatic installer)
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In Appearance->Widgets, add Evolution Twitter Timeline widget to any sidebar.
1. Add your Twitter Name and your Twitter ID
